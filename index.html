List of Semantics for HTML

A guide to what HTML semantic tags represent in an HTML document



Semantics Table

Tag Name Purpose
<title> Title Shows the browser tab and search results
<header> Header Can either be the head of the webpage when inside the <body> or the title of an <article>
<main> Main Hold the main content of the webpage
<article> Article An independent piece of content on a webpage that could be a standalone
<aside> Aside Content that is supplementary to the <main> content of the page
<link> Link Represents an element that links to other webpages using href and rel
<footer> Footer The footer of the website when nested within the <body> element and the least important info inside an <article>
<nav> Navigation An element that contains links to other sources or webpages
<section> Section Contains groups of related elements
<figure> Figure Can contain images, code and other visual representations that could standalone
<figcaption> Figure caption Adds a caption to the <figure> element and must be nested inside of it at all instances
